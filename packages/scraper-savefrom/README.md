# Youtube Scraper
Scraper Savefrom

## Installation
```sh
npm i @reyendymion/scraper-savefrom
```

## Usage 
Example download youtube video
```ts
// import module
import { savefrom } from '@reyendymion/scraper-savefrom'

const data = await savefrom('https://youtu.be/iik25wqIuFo')
console.log(data) // JSON
```
